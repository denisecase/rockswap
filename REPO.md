# What's in this Project?

Many of the files and folders in this project repository ("repo") are boilerplate and you won't ever need to look at them.

## Interesting Areas

- ⭐ [`README.md`](./README.md) - How to play the game.
- ⭐ [`CONTRIBUTING.md`](./CONTRIBUTING.md) - How to make changes or run locally.
- ⭐ [`src/`](./src) - The heart of the project (game logic). For an intro, see [`SRC.md`](./SRC.md).
- ⭐ [`index.css`](./index.css) - Change colors, fonts, and layout theme.
